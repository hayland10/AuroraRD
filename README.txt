La aplicación web de "Aurora", desarrollada con .NET 6.0 y el patrón MVC, es una plataforma administrativa para la gestión de eventos y reservas en Quito. Diseñada para administradores de eventos, permite controlar la publicación, edición y eliminación de eventos, así como gestionar reservas de usuarios. Con una interfaz intuitiva, proporciona análisis y reportes detallados, mejorando la toma de decisiones. Está integrada con una API Web Local y vinculada a una base de datos SQL local, asegurando una gestión de datos coherente y segura entre la aplicación web y móvil.
¿Cómo se usa?
Por el momento solamente se usa localmente a través de VisualStudio debido a que se tiene únicamente una base de datos alojada localmente en SQL Server
Nota:
Para correr la aplicación se debe tener corriendo la API debido a que es una API con base de datos local
